# Schedule OGU
Мобильное приложение (Android) для просмотра расписания занятий ОГУ им. Тургенева

## Сборка
Установленное ПО:
- NodeJS v20.4.0
- Apache Cordova 12.0.0
- Android SDK
    - platforms;android-33
    - build-tools;33.0.2

1. Установить все плагины и пакеты
    ```cordova prepare```
2. Собрать проект
    ```cordova build android --release```
